## Installation & Setup
Download and extract the project files.
   
### **Windows Users**
Open a terminal in the project's folder and run
   ```sh
   ./main.bat
  ```

### **Linux Users**
Open a terminal in the project's folder and run
   ```sh
   python3 -m pip install -r requirements.txt
  ```
